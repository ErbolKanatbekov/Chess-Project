# Simple Chess Web-App

Интерактивное веб-приложение для игры в шахматы, реализованное с использованием современных JavaScript-библиотек. Проект позволяет разыгрывать партии в браузере, соблюдая все официальные правила шахмат.

## 🚀 Функционал
* **Валидация ходов**: Интеграция с `chess.js` гарантирует соблюдение правил (рокировка, взятие на проходе, превращение пешки).
* **Подсчет преимущества**: В реальном времени вычисляется материальный перевес сторон на основе стандартных весов фигур.
* **История взятий**: Визуальное отображение всех съеденных в процессе игры фигур.
* **Умный интерфейс**: Блокировка ходов при окончании игры и предотвращение перемещения фигур противника в его ход.
* **Уведомления**: Оповещение игрока при постановке мата.

## 🛠 Технологии
* [Chessboard.js](https://chessboardjs.com/) — визуализация шахматной доски.
* [Chess.js](https://github.com/jhlywa/chess.js) — шахматная логика и валидация.
* [jQuery](https://jquery.com/) — вспомогательная библиотека для работы с DOM и Chessboard.js.
* **HTML5/CSS3** — структура и оформление.
